# Shift — Le Hackathon Gen AI à Nantes

Source code of [**shift-hackathon.com**](https://shift-hackathon.com), the website of **Shift**, a Gen AI hackathon held in Nantes, France.

> 48 hours to build a game-changing Gen AI product. — *Time to Custom: hack your favorite tool.*

## Repository layout

The repo holds the current site alongside archived editions, side by side:

Each new edition backs up the previous site into a `202x/` directory before the current one rolls over.

Issues and pull requests are welcome. Before opening a PR:

